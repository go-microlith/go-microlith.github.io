Endpoints

Endpoints are Lambda functions triggered on HTTP requests. Create an Endpoint Lambda by implementing the Handler interface.

Within the handler Lambda functions, you can parse a RequestBody() as JSON and Respond() with JSON or an error. The path and query parameters and the request headers are available on the request object.

type echo struct {}

func (*echo) ServeREST(ctx context.Context, req events.APIGatewayProxyRequest) (events.APIGatewayProxyResponse, error) {
  var body map[string]interface{}
  if err := rest.RequestBody(&body); err != nil {
    return rest.Respond(http.StatusBadRequest, err)
  }

  return rest.Respond(http.StatusOK, body)
}

APIs

Define APIs on services by calling the API() method on a service builder. Define the origin API on a domain by calling the Origin() method on a domain builder. In the root domain, use the microlith.Origin() function.

In the builder, you can add Head(), Get(), Post(), Put(), and Delete() endpoints and define Scope()s. The HTTP methods take a unique endpoint name, an Endpoint Lambda, and an optional output Endpoint argument.

var echoEndpoint *rest.Endpoint

service.API(func(api *rest.APIBuilder) {
  api.Post("echo", new(echo), &echoEndpoint)
})

The HTTP methods take an optional builder to define the Query() parameters and request Header()s, as well as the Response()s and response Header()s.

var listOrdersEndpoint *rest.Endpoint

service.API(func(api *rest.APIBuilder) {
  api.Get("list-orders", new(listOrders), &listOrdersEndpoint, func(endpoint *rest.EndpointBuilder) {
    endpoint.Query("page", false)
    endpoint.Query("limit", false)

    endpoint.Header("Authorization", true)

    endpoint.Response(http.StatusOK, func(response *rest.ResponseBuilder) {
      response.Header("X-Next-Page", false)
    })
  })
})

In Lambda functions, you can Call() the endpoint, passing an optional request body. The returned values are the same values returned by http.Client. You can decode the ResponseBody() as JSON, or you can get the ErrorResponse() if an error was passed to Respond().

resp, err := echoEndpoint.Call(ctx, map[string]interface{}{"Hello": "World"})
if err != nil {
  // ...
}

switch resp.StatusCode {
case http.StatusOK:
  response := make(map[string]interface{})
  if err := rest.ResponseBody(resp, &response); err != nil {
    // ...
  }
  fmt.Printf("Response: %#v\n", response)
default:
  err := rest.ErrorResponse(resp, nil)
  fmt.Printf("Error: %s\n", err)
}

You can use an optional builder to set the Query() and Path() parameters and request Header()s.

resp, err := listOrdersEndpoint.Call(ctx, nil, func(request *rest.RequestBuilder) {
  request.Query("page", "2")
  request.Query("limit", "10")

  request.Header("Authorization", "<oauth-token>")
})

Scopes

Scopes are HTTP paths. They can be multi-segment and contain path parameters in the form “{parameter-name}”. Scopes are created by calling Scope() on an API builder. Scopes can be nested within other scopes by calling Scope() on a scope builder.

service.API(func(api *rest.APIBuilder) {
  api.Scope("/orders", func(scope *rest.ScopeBuilder) {
    scope.Get("list-orders", new(listOrders), nil)
    scope.Post("create-order", new(createOrder), nil)

    scope.Scope("/{order-id}", func(scope *rest.ScopeBuilder) {
      scope.Get("get-order", new(getOrder), nil)
      scope.Put("update-order", new(updateOrder), nil)
      scope.Delete("delete-order", new(deleteOrder), nil)
    })
  })
})

Query() parameters, request Header()s, and Response()s can be defined on a scope and will be inherited by all endpoints in that scope and all nested scopes. Inherited values can be overridden in individual endpoints or nested scopes.

service.API(func(api *rest.APIBuilder) {
  api.Scope("/orders", func(scope *rest.ScopeBuilder) {
    scope.Header("Authorization", true)
    scope.Response(http.StatusInternalServerError)

    scope.Get("list-orders", new(listOrders), nil, func(endpoint *rest.EndpointBuilder) {
      endpoint.Header("Authorization", false)

      endpoint.Response(http.StatusOK, func(response *rest.ResponseBuilder) {
        response.Header("X-Next-Page", false)
      })
    })

    scope.Post("create-order", new(createOrder), nil, func(endpoint *rest.EndpointBuilder) {
      endpoint.Response(http.StatusCreated)
      endpoint.Response(http.StatusBadRequest)
    })

    scope.Scope("/{order-id}", func(scope *rest.ScopeBuilder) {
      scope.Response(http.StatusOK)
      scope.Response(http.StatusBadRequest)
      scope.Response(http.StatusNotFound)

      scope.Get("get-order", new(getOrder), nil)
      scope.Put("update-order", new(updateOrder), nil)
      scope.Delete("delete-order", new(deleteOrder), nil)
    })
  })
})
